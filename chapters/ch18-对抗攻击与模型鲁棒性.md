# Chapter 18: 对抗攻击与模型鲁棒性

## Core Idea
深度学习模型虽高正确率,但对**恶意扰动**极其脆弱--人眼无法察觉的微小噪声能让 50 层 ResNet 把虎斑猫识别为海星(置信度 1.0)。**对抗攻击**研究如何构造这种扰动:**白盒攻击**已知模型参数算梯度,**黑盒攻击**用代理网络模仿后攻击。**FGSM(快速梯度符号法)**只需一次迭代,取梯度符号乘 ε 即可。防御分两类: **被动防御**在模型前加滤波器(模糊化、压缩、生成、随机化);**主动防御**用对抗训练产生鲁棒模型。攻防仍在不断演化,攻击一般比防御容易。

## Frameworks Introduced

- **对抗攻击优化问题 (LeeDL)**
  - When to use: 攻击固定模型
  - How:
    - **无目标攻击**: 找 x 最小化 `L(x) = -e(y, ŷ)`(让输出与正确标签越远越好)
    - **有目标攻击**: `L(x) = -e(y, ŷ) + e(y, y^target)`(远离正确 + 接近目标)
    - 约束: `d(x₀, x) ≤ ε`(人眼不可察觉)
  - 关键: 把输入当参数,固定网络用梯度下降调输入

- **距离度量 L2 vs L∞ (LeeDL)**
  - When to use: 衡量扰动大小
  - How:
    - **L2**: `||Δx||₂ = √(ΣΔxᵢ²)`(整体变化)
    - **L∞**: `||Δx||∞ = max(|Δxᵢ|)`(最大单维变化)
  - **L∞ 更符合人眼感知**: 人眼关注最大变化量,不是平均
  - 约束用 L∞ 时,x 必须落在以 x₀ 为中心的方形框内

- **FGSM 快速梯度符号法 (LeeDL)**
  - When to use: 快速生成对抗样本
  - How:
    - 计算梯度 g = ∇ₓ L(x)
    - 取符号: g = sign(g) ∈ {-1, +1}
    - 学习率 η = ε
    - 更新: `x' = x - ε · sign(g)`(一次迭代)
  - 关键: 更新后 x' 一定落在约束方框的**四个角落**
  - 变体: I-FGSM(迭代版)效果更好但更复杂

- **白盒攻击 vs 黑盒攻击 (LeeDL)**
  - When to use: 区分攻击者掌握的信息
  - How:
    - **白盒**: 知道模型参数、结构、梯度,直接算梯度攻击
    - **黑盒**: 不知道参数,**训代理网络**模仿:
      - 有训练数据: 用同数据训代理模型,攻击代理模型迁移到原模型
      - 无训练数据: 查询原模型(输入输出对)训代理模型
  - 关键: 黑盒攻击易成功,因为攻击方向对不同网络相似--"问题来自数据而非模型"

- **集成黑盒攻击 (LeeDL)**
  - When to use: 提高黑盒攻击成功率
  - How: 用多个代理网络(如 5 个不同架构),**联合生成对抗扰动**攻击所有代理网络。对角线(被攻击模型不在代理集合中)黑盒正确率 < 6%
  - 关键: 集成让攻击方向更"通用",跨网络迁移性更强

- **特殊攻击类型 (LeeDL)**
  - When to use: 不同场景
  - How:
    - **单像素攻击**: 只改一个像素(局限性大,成功率低)
    - **通用对抗攻击**: 一个扰动攻击所有图片(可部署在监视器摄像头)
    - **现实世界攻击**: 眼镜骗人脸识别、改交通标志牌(考虑多角度、物理可实现)
    - **对抗重编程**: 寄生已有模型做不同任务(ImageNet 模型数方块)
    - **后门攻击**: 训练阶段植入后门,测试时特定输入触发误判,数据看似正常

- **被动防御 (LeeDL)**
  - When to use: 已训好的模型不想重训
  - How: 模型前加滤波器,削减攻击信号:
    - **模糊化**: 轻微平滑,破坏攻击噪声(副作用: 置信度下降)
    - **图像压缩**: JPEG 失真破坏攻击信号
    - **生成器重构**: 让生成器复现图片,攻击噪声难以复现
    - **随机化防御**: 随机缩放、平移、贴灰背景--欲欺敌先瞒内
  - **致命弱点**: 攻击者知道防御方式则失效(可把滤波器当第一层纳入攻击优化)

- **主动防御 / 对抗训练 (LeeDL)**
  - When to use: 训练阶段提升鲁棒性
  - How:
    - 训练初始模型
    - 对每张训练图 x 生成对抗样本 x̃
    - 给 x̃ 标正确标签 ŷ
    - 用 (X ∪ X') 联合训练
  - 关键: 本质是**数据增强**,即使无攻击也能提升泛化
  - **缺点**: ① 挡不住新攻击算法(训时没考虑);② 计算成本巨大(每张图都要攻击)

## Key Concepts

- **对抗攻击 (adversarial attack)**: 加微小噪声骗网络
- **对抗样本 (adversarial example)**: 加噪后的图片
- **无目标攻击 (untargeted attack)**: 输出 ≠ 正确即可
- **有目标攻击 (targeted attack)**: 输出 = 指定错误类别
- **白盒攻击 (white-box)**: 已知模型参数
- **黑盒攻击 (black-box)**: 未知参数,用代理网络
- **代理网络 (proxy network)**: 模仿被攻击网络的替代模型
- **FGSM**: 快速梯度符号法,一次迭代
- **I-FGSM**: 迭代版 FGSM
- **L∞ 范数约束**: 最大单维变化,符合人眼感知
- **单像素攻击**: 只改一个像素
- **通用对抗扰动**: 一个扰动通杀所有图片
- **对抗重编程**: 寄生模型做不同任务
- **后门攻击 (backdoor)**: 训练阶段植入
- **被动防御 (passive defense)**: 加滤波器
- **主动防御 (active defense)**: 对抗训练
- **模糊化 (smoothing)**: 被动防御一种
- **随机化防御 (randomization)**: 随机变换
- **对抗训练 (adversarial training)**: 用对抗样本增强训练数据

## Mental Models

- **攻击 = 调输入而非参数**: 固定网络,梯度下降找最坏输入
- **L∞ 方框约束**: 更新后 x 必须在方形框内,出界就拉回
- **FGSM 一步到角落**: 符号函数让更新后 x' 落在方框四角
- **黑盒攻击 = 代理网络模仿**: 同数据训代理,攻击迁移
- **问题来自数据**: 不同网络攻击方向相似,说明数据本身特征如此
- **被动防御 = 盾牌**: 一次过滤,被识破就失效
- **主动防御 = 免疫训练**: 把攻击样本喂给模型,学会抵抗
- **攻易守难**: 攻击者只需找到一个漏洞,防御者要挡所有攻击

## Anti-patterns

- **用 L2 衡量图像扰动**: 不符合人眼感知。**用 L∞**
- **FGSM 多次迭代**: 易出界。**一次即可,或用 I-FGSM 带约束**
- **白盒假设模型不公开就安全**: 黑盒攻击可成功。**必须做防御**
- **单一被动防御**: 被识破就失效。**用随机化或主动防御**
- **模糊化太强**: 正常图也识别错。**轻微模糊即可**
- **对抗训练只用一种攻击算法**: 挡不住新攻击。**多种攻击算法混合训练**
- **不检查公开数据集**: 可能有后门。**审查数据来源**
- **期望对抗训练挡所有攻击**: 新攻击算法可能突破。**持续更新训练攻击**
- **黑盒有目标攻击**: 跨网络迁移性差。**优先无目标攻击**

## Code Examples

**FGSM 攻击**:

```python
import torch
import torch.nn.functional as F

def fgsm_attack(model, x, y_true, epsilon, targeted=False, y_target=None):
    """FGSM: 一次迭代生成对抗样本
    x: 原图 (B, C, H, W), requires_grad=True
    y_true: 真实标签
    epsilon: 扰动大小 (L∞ 约束)
    """
    x.requires_grad = True
    output = model(x)
    
    if targeted:
        # 有目标: 最小化 e(y, y_target) + 最大化 e(y, y_true)
        loss = -F.cross_entropy(output, y_true) + F.cross_entropy(output, y_target)
    else:
        # 无目标: 最大化 e(y, y_true) = 最小化 -e(y, y_true)
        loss = -F.cross_entropy(output, y_true)
    
    model.zero_grad()
    loss.backward()
    
    # 取梯度符号,一步更新
    perturbation = epsilon * x.grad.sign()
    x_adv = x - perturbation  # 最小化 loss = 最大化交叉熵
    
    # L∞ 约束 clip
    x_adv = torch.clamp(x_adv, x - epsilon, x + epsilon)
    x_adv = torch.clamp(x_adv, 0, 1)  # 像素值范围
    return x_adv.detach()

# 使用
model.eval()
for x, y in dataloader:
    x_adv = fgsm_attack(model, x, y, epsilon=0.03)
    print("原图预测:", model(x).argmax(1))
    print("对抗样本预测:", model(x_adv).argmax(1))
```

**I-FGSM(迭代版)**:

```python
def i_fgsm(model, x, y_true, epsilon, alpha=0.01, num_iter=10):
    """I-FGSM: 多次小步迭代,每步约束在 L∞ 球内"""
    x_adv = x.clone().detach()
    for _ in range(num_iter):
        x_adv.requires_grad = True
        output = model(x_adv)
        loss = -F.cross_entropy(output, y_true)  # 无目标
        model.zero_grad()
        loss.backward()
        # 小步走
        x_adv = x_adv - alpha * x_adv.grad.sign()
        # 投影回 L∞ 球
        x_adv = torch.clamp(x_adv, x - epsilon, x + epsilon)
        x_adv = torch.clamp(x_adv, 0, 1).detach()
    return x_adv
```
- **What it demonstrates**: I-FGSM 每步用小 alpha,多次迭代更精细,但每步都约束

**黑盒攻击(代理网络)**:

```python
# 1. 训练代理网络(用原模型的输入输出对)
proxy = ResNet18()
proxy_opt = torch.optim.Adam(proxy.parameters())

for x, _ in dataloader:
    with torch.no_grad():
        y_soft = target_model(x)  # 原模型(黑盒)输出
    # 训练代理网络模仿原模型
    proxy_out = proxy(x)
    loss = F.kl_div(proxy_out.log(), y_soft, reduction='batchmean')
    proxy_opt.zero_grad(); loss.backward(); proxy_opt.step()

# 2. 对代理网络做白盒攻击
x_adv = fgsm_attack(proxy, x, y, epsilon=0.03)

# 3. 用对抗样本攻击原模型
print("原模型对 x_adv 的预测:", target_model(x_adv).argmax(1))
```

**被动防御(模糊化)**:

```python
import torch.nn as nn

class SmoothingDefense(nn.Module):
    """模型前加高斯模糊滤波器"""
    def __init__(self, model, kernel_size=3, sigma=0.5):
        super().__init__()
        self.model = model
        self.blur = nn.GaussianBlur(kernel_size, sigma)
    
    def forward(self, x):
        x_smoothed = self.blur(x)
        return self.model(x_smoothed)

# 包装防御
defended_model = SmoothingDefense(resnet, kernel_size=3, sigma=0.5)
# 对抗样本被模糊化后失效
```

**对抗训练(主动防御)**:

```python
def adversarial_training(model, x, y, optimizer, epsilon=0.03):
    """一步对抗训练: 生成对抗样本 + 用对抗样本训模型"""
    # 1. 生成对抗样本
    x.requires_grad = True
    output = model(x)
    loss = F.cross_entropy(output, y)
    loss.backward()
    x_adv = (x + epsilon * x.grad.sign()).detach()
    x_adv = torch.clamp(x_adv, 0, 1)
    
    # 2. 用对抗样本(带原标签)训模型
    optimizer.zero_grad()
    pred = model(x_adv)
    loss_adv = F.cross_entropy(pred, y)
    loss_adv.backward()
    optimizer.step()
    return loss_adv.item()

# 训练循环
for epoch in range(num_epochs):
    for x, y in dataloader:
        loss = adversarial_training(model, x, y, optimizer)
```

## Worked Example

**为什么 FGSM 一次迭代就能成功攻击?** -- 虎斑猫 → 海星案例。

**情景**: 50 层 ResNet 对虎斑猫图片输出 0.64 置信度。攻击者想把它变成海星(置信度 1.0)。

**传统梯度下降**(多次迭代):
```
x₀ → ∇L → x₁ → ∇L → x₂ → ... → x_n
```
每次小步,慢。

**FGSM 关键洞察**:
1. **取符号而非实际值**: `g = sign(∇ₓL)` ∈ {-1, +1}ᵈ
   - 只保留"该往哪个方向变"的信息,丢弃大小
2. **学习率 = ε**(L∞ 约束边界)
   - 一步走到约束框的角落
3. **结果**: `x' = x - ε · sign(∇ₓL)`
   - 更新后 x' 在方框四角之一
   - 每个像素最多变化 ε

**为什么有效**:
1. **线性假设**: Goodfellow 2014 解释--高维空间中,即使每维扰动小(ε=0.03),累积起来足以改变输出。线性模型就有此问题,深层网络更甚
2. **方向比大小重要**: 符号函数保留方向,足以推向决策边界
3. **方框角落是最优点**: 在 L∞ 球内,使 loss 最大的点必在角落(线性目标函数)

**LeeDL 实验数据**:
- 原图: 虎斑猫, 置信度 0.64
- FGSM 攻击后(ε 肉眼不可见): 海星, 置信度 1.0
- **人眼无法分辨原图和对抗样本**

**为什么黑盒攻击也能成功?**
- 实验观察: 让 VGG-16 攻击成功的方向,对 ResNet/GoogLeNet 也大概率有效
- **结论**: 攻击方向跨网络迁移--问题来自数据特征,不是模型
- **数据决定边界**: 训练数据决定所有模型的决策边界相似,因此一个模型的"漏洞"也是其他模型的

**关键洞察**:
1. **FGSM 极简**: 符号 + 一步 = 攻击成功
2. **L∞ 约束自然满足**: 学习率 ε 直接对应约束边界
3. **攻击易防御难**: 攻击找一个方向即可,防御要挡所有方向
4. **黑盒可行**: 数据决定攻击迁移性,代理网络能逼近原模型
5. **被动防御有上限**: 一旦攻击者知道防御方式,可纳入攻击优化破解
6. **对抗训练是最强防御**: 但计算成本巨大,且挡不住训练时未考虑的新攻击

## Key Takeaways

1. 对抗攻击 = 加微小噪声(人眼不可见)让网络输出错误结果,ResNet 50 也难幸免
2. 攻击优化问题: 固定模型参数,梯度下降调输入,**约束 d(x₀, x) ≤ ε**
3. **L∞ 范数**比 L2 更符合人眼感知(关注最大变化而非平均)
4. **FGSM** = 梯度符号 + 一次迭代 + 学习率 ε,极简高效的攻击
5. **白盒攻击**: 已知参数直接算梯度;**黑盒攻击**: 训代理网络,攻击迁移
6. 黑盒攻击易成功因为"问题来自数据"--不同网络攻击方向相似
7. **集成攻击**: 多代理联合,黑盒正确率 < 6%
8. 特殊攻击: 单像素、通用扰动、现实世界(眼镜/路标)、对抗重编程、后门攻击
9. **被动防御**: 模型前加滤波器(模糊化、压缩、生成、随机化),被识破即失效
10. **对抗训练**(主动防御): 用对抗样本训模型,最强但计算成本巨大,挡不住新攻击算法
